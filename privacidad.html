Política de Privacidad

Última actualización: 27 de mayo de 2026

1. Responsable del tratamiento

Ricard Penin Honrubia
Correo electrónico: ergoegos@gmail.com

2. Datos que recogemos

  • Formulario de contacto: nombre, correo electrónico, tipo de servicio y mensaje. Se envían a través del servicio FormSubmit.co para que pueda responderte.
  • Reserva de consulta: al usar el calendario integrado (Cal.com), se recogen los datos que introduzcas en su plataforma. Cal.com actúa como encargado del tratamiento bajo su propia política de privacidad.
  • Almacenamiento local (localStorage): se guarda tu preferencia de tema visual (claro/oscuro). Este dato no sale de tu navegador y no te identifica.
  • Caché de sesión (sessionStorage): se almacena temporalmente información pública de repositorios de GitHub para acelerar la navegación. Se borra al cerrar la pestaña.

3. Finalidad

Los datos del formulario se utilizan exclusivamente para responder tu consulta y, si lo solicitas, ofrecerte un presupuesto. No se usan para marketing ni se venden a terceros.

4. Base legal

El tratamiento se basa en tu consentimiento explícito (artículo 6.1.a del RGPD), que otorgas al marcar la casilla del formulario de contacto.

5. Terceros que reciben datos

  • FormSubmit.co — procesa el envío del formulario de contacto.
  • Cal.com — gestiona la reserva de consultas (se carga bajo demanda).
  • GitHub API — se consulta la API pública de GitHub para mostrar información de repositorios. Tu navegador envía su dirección IP a los servidores de GitHub al acceder a la sección de portfolio.

6. Cookies y almacenamiento

Este sitio no utiliza cookies. El único almacenamiento en tu dispositivo es:

  • localStorage: preferencia de tema (funcional, no requiere consentimiento).
  • sessionStorage: caché temporal de la API de GitHub (se elimina al cerrar la pestaña).
  • Service Worker: caché de archivos del propio sitio para funcionamiento offline.

7. Conservación

Los datos del formulario se conservan únicamente el tiempo necesario para responder tu consulta. Puedes solicitar su eliminación en cualquier momento.

8. Tus derechos

Tienes derecho a:

  • Acceder a tus datos personales.
  • Rectificar datos inexactos.
  • Solicitar la supresión de tus datos.
  • Oponerte al tratamiento.
  • Solicitar la portabilidad de tus datos.
  • Limitar el tratamiento.

Para ejercer cualquiera de estos derechos, escríbeme a ergoegos@gmail.com.

9. Reclamaciones

Si consideras que tus derechos no han sido atendidos correctamente, puedes presentar una reclamación ante la Agencia Española de Protección de Datos (AEPD).
